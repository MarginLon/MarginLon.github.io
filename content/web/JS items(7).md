# 块级作用域
[原文章](https://juejin.im/post/6844903955814694919)

- 块级作用域内默认变量
    - 不带var，let，const，只有执行过定义的变量的代码后才可以访问，给window赋值属性，之前会报错
    - 块内默认变量依旧是全局变量
    - 没执行之前不可以访问
- 块级作用域函数声明
    - 块内函数声明会提升到块顶部，也会在全局作用域用var声明一个同名的undefined变量
    - 块外的全局同名变量的赋值时机是执行完块内函数声明语句
